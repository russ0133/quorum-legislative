# quorum-legislative

Steps to run:

Clone the repository

- git clone git@github.com:russ0133/quorum-legislative.git
- cd quorum-legislative

Start the Node server

- cd server
- npm i ../shared
- npm run dev

Start the React client

- cd ..
- cd client
- npm i ../shared
- npm run dev

Access the server at http://localhost:5173/
